module steps

Script for finding configurations

When called directly as main, it will provide the 'scores' of all the possible configurations. 

$ python -m mlcsim.steps --help

usage: steps.py [-h] [-b {2,3,4}] [-c {2,3,4,5,6,7,8}] --thr THR

options:

    -h, --help          show this help message and exit

    -b {2,3,4}          bits per cell

    -c {2,3,4,5,6,7,8}  num of cells  --thr THR           Threshold map JSO

Prints out a pretty markdown table
