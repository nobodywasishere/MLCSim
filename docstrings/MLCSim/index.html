module MLCSim

MLCSim class

This module provides the MLCSim class, which is used for encoding and decoding values to and from its MLC configuration.

When called directly as main, it allows for encoding and decoding a value using a cell configuration json. 

$ python -m mlcsim.MLCSim --help

usage: MLCSim.py [-h] -f F {enc,dec} val

positional arguments:
   {enc,dec}   action to take on value
   val         val to {en,de}code

options:

    -h, --help  show this help message and exit

    -f F        cell config json
